# 사용자 맞춤형 온수난방 시스템 개발

## Overview

기존 온수·난방 제어 시스템과 연동되는 Spring 기반 웹 서비스와 관리자 대시보드를 개발한 프로젝트입니다.  
IoT 기기 통신, FOTA, 실시간 이벤트 전달, 데이터 처리 파이프라인을 구현했습니다.

- **기간:** 2023.10 ~ 2026.01
- **개발 기간:** 2023.10 ~ 2024.05
- **참여 인원:** 2명
- **역할:** 백엔드 API 개발, IoT 통신 기능 구현, 실시간 데이터 처리, 관리자 대시보드 개발
- **사용기술:** Java, Spring Boot, Vue.js, MySQL, Kafka, MQTT, Firebase, FOTA, R, Ubuntu, CentOS

## Problem

- 기기 펌웨어를 직접 전송하는 구조에서 전송 지연, 기기 오작동, 다운로드 실패 문제가 발생
- IoT 기기에서 발생하는 데이터와 이벤트를 안정적으로 수집하고, 운영자가 상태를 확인할 수 있는 구조가 필요

## Solution

### 1. 서버 중계 다운로드 방식 전환

대용량 펌웨어를 직접 전송하지 않고 서버 중계 다운로드 방식으로 전환해 기기에서 서버로 API를 전송하여 펌웨어 파일을 내려받도록 하여 배포 안정성 확보

### 2. MQTT 기반 기기 통신

MQTT를 활용해 기기와 서버 간 통신 구조를 구현하여, 기기 상태와 이벤트를 안정적으로 전달할 수 있도록 구성

### 3. Kafka 기반 데이터 처리

Kafka 기반 실시간 데이터 처리 파이프라인을 구축해 IoT 데이터를 안정적으로 스트리밍 처리

### 4. 관리자 대시보드와 알림

Vue.js 관리자 대시보드를 개발하고 Firebase 기반 앱 알림 기능을 구현해 실시간 이벤트 전달과 모니터링 기능을 제공

## Result

- 펌웨어 배포 안정성 확보
- MQTT 기반 기기 통신 및 FOTA 기능 구현
- Kafka 기반 IoT 데이터 스트리밍 처리 구조 구축
- Firebase 기반 실시간 이벤트 알림 제공
- Vue.js 관리자 대시보드를 통한 모니터링 및 제어 기능 제공
